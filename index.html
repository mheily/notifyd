stated - a state notification daemon

stated (pronounced "state dee") is a mechanism for unrelated programs to communicate about changes to their internal state.

Getting the source code

The source code is available on the GitHub project page at https://github.com/mheily/stated

You can checkout a copy of the source code by running:

git clone https://github.com/mheily/stated.git

Usage

See the state.h page for the API documentation.

There are also example programs that demonstrate typical use cases; see the Examples page.
